# 🏠 Utility Usage Prediction Tool (ML Console)

A Python-based console application developed as part of my **AI/ML Internship at CodeVedX**. This project helps users manage utility usage records, store data in CSV files, and predict future utility usage using a simple machine learning model.

## 📖 Project Overview

The Utility Usage Prediction Tool is a menu-driven console application designed to:

- Manage utility usage records
- Store and retrieve data using CSV files
- Predict utility usage based on historical data
- Demonstrate Python programming and basic machine learning concepts

This project focuses on applying Python fundamentals, data handling, and introductory ML techniques in a real-world scenario.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/gokila-12/Utility-Usage-Prediction.git
```

### Navigate to the Project Folder

```bash
cd Utility-Usage-Prediction
```

### Install Required Libraries

```bash
pip install numpy
```

### Run the Application

```bash
python utility_usage-prediction.py
```
