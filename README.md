## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/gokila-12/CODEVEDX.git
```

Navigate to any project folder:

```bash
cd student-performance-prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
