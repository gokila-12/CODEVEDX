# 🎓 Student Performance Prediction System

A Machine Learning-based console application that predicts a student's final performance using attendance, marks, and study hours. The project also includes data preprocessing, visualization, and model evaluation to provide insights into student performance.

## 📌 Project Overview

This project demonstrates a complete Machine Learning workflow:

- Load and preprocess student data
- Handle missing values
- Perform Exploratory Data Analysis (EDA)
- Train a Machine Learning model
- Predict student performance
- Evaluate model accuracy
- Visualize data using charts

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/gokila-12/student performance prediction.git
```

### 2. Navigate to the Project Folder

```bash
cd student performance prediction
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python main.py
```

## 📋 Workflow

1. Load the student dataset.
2. Preprocess and clean the data.
3. Handle missing values.
4. Perform data visualization.
5. Train the Machine Learning model.
6. Evaluate model performance.
7. Enter attendance, marks, and study hours.
8. Predict the student's final performance.
9. Display the prediction result.
